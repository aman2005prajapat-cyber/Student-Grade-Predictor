 # 🎓 StudySync: Student Grade Predictor

StudySync is a web application built with **Flask** that predicts a student's final exam score based on their study habits. 

## 🚀 Features
* **Instant Prediction:** Get immediate score estimates based on study hours.
* **Clean UI:** Simple and responsive web interface.
* **Flask Backend:** Demonstrates how to handle user data and return results.

## 🛠️ Tech Stack
* **Python** (Backend Logic)
* **Flask** (Web Framework)
* **HTML/CSS** (Frontend)

## 💻 How to Run Locally
1. Clone the repository:
   ```bash
   git clone [https://github.com/aman2005prajapat-cyber/Student-Grade-Predictor.git](https://github.com/aman2005prajapat-cyber/Student-Grade-Predictor.git)
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
    python app.py
